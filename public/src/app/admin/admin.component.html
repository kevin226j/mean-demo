Dashboard Information:

  1. After the instance of logging in, this dashboard will expire after one full day.
  2. Right side panel will allow you to navigate between each manageable component. Note that, there is no auto-save; meaning if a new tab is clicked, informaiton stored on a previous tab will be lost.
  3. This dashboard enables the user to edit the following sections:
    • Tag: allowing full access to add, update, and delete a tag. Note: if an album does not contain a tag, it will NOT apear on drop-down, or filter component.
      Edit the Tags Now!
    • Album: allowing user to customize an album. It is recommended to create an album as the following step from the tag section above. If tag is already in place, proceed in this section.
      Edit the Albums Now!
    • Photos: allowing user to preview photos to upload before adding them to selected album. Note, there is some wait time when viewing current photos as its pulling data from multiple resources.
      Edit the Photos Now!
